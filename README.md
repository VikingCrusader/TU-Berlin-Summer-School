## 🛠️ Technologies & Concepts Covered

**Core Java**
- Object-Oriented Programming (OOP): inheritance, polymorphism, encapsulation, abstraction
- Collections Framework, Generics, Lambda expressions
- Exception handling, File I/O
- Multithreading and concurrency (introductory)

**GUI & Database**
- Swing GUI development and event-driven programming
- JDBC and SQLite/MySQL database connectivity
- ORM concepts: Hibernate / JPA (introductory)

**Web & Frameworks**
- Spring Framework and Dependency Injection (introductory)
- RESTful Web Services concepts (introductory)
- Servlets and JSP (introductory)

**Tools & Collaboration**
- Git and GitHub for collaborative development (branching, merging)
- Maven for project management
- Eclipse IDE

## 📁 Projects

### Project 1 – Shipment Management
A shipment tracking system built with Java and OOP principles.  
**Key concepts:** Classes, inheritance, encapsulation, collections

### Project 2 – War Game
A turn-based strategy game implementing game logic with Java.  
**Key concepts:** OOP design, event handling, Swing GUI

### Project 3 – Dental Appointment System ⭐
A dental reservation system with persistent data storage.  
**Key concepts:** JDBC, SQLite, Maven, refactoring  
👉 *Refactored and extended version:* [Dental-Reservation-System](https://github.com/VikingCrusader/Dental-Reservation-System)
